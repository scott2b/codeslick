World of Need

WorldOfNeed.org is a site built on a foundation of content aggregation and automated narrative content generation. Utilizing data available from the CIA and other government sources, I was able to rapidly generate per-country narrative content about the various states of poverty, sickness, education, etc. in the world. Further content for the site is was gathered from Kiva.org, a project that enables microlending to empower those in need. (Note: To conserve resources, Kiva content gathering is currently disabled on the site).

Features

  • automatically generated narrative content
  • aggregated content from the Kiva API.

Technologies

  • Python
  • Django
  • PostgreSQL
  • MongoDB

Contributions

  • Full web application development with automated content generation and aggregation tier
  • Visual design provided by a template from themeforest.net
