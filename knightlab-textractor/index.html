Knightlab Textractor

I built a text extractor web service for the Knight Lab. This service is a web front-end to the Java Boilerpipe text extraction work by Christian Kohlsch√ºtter. Technologies used include:

  • Java Boilerpipe
  • python-boilerpipe
  • Django 1.5
  • John Resig’s Javascript file diff utility

Text extraction from web pages is a common requirement for many Knight Lab projects. This web service provides a simple means to extract the primary text from the page indicated by URL, and returns the results in JSON format. A user-facing component also allows for side-by-side diff comparison of content extraction using different extraction algorithms available in Boilerpipe.
